les différents fichiers de logs seront positionnés ici :

cache.log      : log relatif au action sur le cache
config.log     : log relatif au configuration
debuglog.log   : log relatif au erreur avec stack
initialize.log : log relatif à la procédure de démarrage du server
io_access.log  : log relatif sur les IO
metrics.log    : log relatif au envois de mesure dans des bases de données types Graphite 
plugins.log    : log relatif à la détection des plugins
provider.log   : log relatif aux informations ou erreur des providers
sse.log        : log relatif aux envois SSE vers les fronts
xllog.log      : log d'erreur prioritaires
